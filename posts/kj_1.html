  • 日期：2021-08-13

Java开发环境配置

一、开发需要用到的工具：

  • IDEA(开发工具)
  • JDK(Java开发环境)
  • MySQL(数据库环境)
  • Redis(数据缓存环境)
  • Maven(依赖管理环境)

二、详细介绍

  • IDEA
    下载地址：https://www.jetbrains.com/zh-cn/idea/download/#section=mac
    破解教程：https://www.exception.site/essay/how-to-free-use-intellij-idea-2019-3

  • JDK
    版本：1.8
    下载地址：https://www.oracle.com/cn/java/technologies/javase/javase-jdk8-downloads.html

  • MySQL
    版本：5.7+ （8.0大变更）
    下载地址：https://dev.mysql.com/downloads/mysql/
    破解教程：http://www.itongji.cn/detail?type=99992840

  • Redis <key,value>形式的数据缓存
    下载地址：https://redis.io
    Redis桌面管理器下载地址：https://rdm.dev
    配置教程：https://blog.csdn.net/weixin_44672591/article/details/105001784

    • 守护进程的方式运行：https://www.runoob.com/redis/redis-conf.html
    • Redis在后台运行：https://blog.csdn.net/gb4215287/article/details/78609918

  • Maven
    通过pom.xml，获取整合jar包，处理兼容
    下载地址：https://maven.apache.org/download.cgi
    配置教程：https://www.cnblogs.com/helios-fz/p/11061033.html
    命令：mvn clean
    打包命令：mvn clean install -Pprod
